Description

This is a utility for the role-playing game Pokemon Tales. It stores the information on Pokemon moves, natures, and abilities, as well as a pokedex listing information on all 1st generation Pokemon, with possibly more to come. Its main function, however, is generating character sheets for wild Pokemon encounters and trainer battles.

We hope you find this a useful aid to your gaming. You can find the rules manual for Pokemon Tales here. We have also developed the campaign setting of Pensino for your use; you can find it here. If you enjoy this game, you should check out our website to see what else we have that you might like.
